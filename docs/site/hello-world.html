Hello World

Your first Nex program, step by step.

The Simplest Program

Create a file called main.nex with the following content:

def main() -> Unit {
    println("Hello, World!")
}

Run it:

nex main.nex

Output:

Hello, World!

Understanding the Code

  • def main() -> Unit — declares the program entry point. Every Nex application needs a main function that returns Unit (void).
  • println("Hello, World!") — prints a string followed by a newline. println is a built-in function available without imports. It accepts multiple comma-separated arguments (like Python's print): println("age:", 25) outputs age: 25.
  • No semicolons needed — Nex uses Automatic Semicolon Insertion (ASI).
  • Braces { } delimit blocks. Indentation is for readability only.

A More Complete Example

Here is a more interesting program demonstrating functions, loops, conditionals, and string operations:

def factorial(n: Int) -> Int {
    if (n <= 1) {
        return 1
    }
    return n * factorial(n - 1)
}

def fizzbuzz(n: Int) -> String {
    if (n % 15 == 0) {
        return "FizzBuzz"
    } else if (n % 3 == 0) {
        return "Fizz"
    } else if (n % 5 == 0) {
        return "Buzz"
    }
    return ""
}

def main() -> Unit {
    println("=== Nex v0.1 ===")

    println("Factorial of 10:")
    println(factorial(10))

    println("FizzBuzz 1-20:")
    i = 1
    while (i <= 20) {
        result = fizzbuzz(i)
        if (result == "") {
            println(i)
        } else {
            println(result)
        }
        i = i + 1
    }

    name = "World"
    println("Hello, " + name + "!")
}

Key Takeaways

  • Variables are declared by first assignment: i = 1 creates an Int variable.
  • Types are inferred from the assigned value — no type annotation needed for locals.
  • String concatenation uses the + operator, and non-string values are auto-converted.
  • Functions declare parameter types and return types explicitly.
  • Recursion works as expected.
