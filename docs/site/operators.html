Operators & Expressions

Complete reference for Nex's operators, precedence, and expression evaluation.

Arithmetic Operators

a = 10 + 3    // 13
b = 10 - 3    // 7
c = 10 * 3    // 30
d = 10 / 3    // 3 (integer division)
e = 10 % 3    // 1 (modulo)

Comparison Operators

a == b    // equal
a != b    // not equal
a < b     // less than
a <= b    // less than or equal
a > b     // greater than
a >= b    // greater than or equal
Note

For classes, == checks reference equality by default unless operator== is overloaded. For structs, == does fieldwise comparison by default.

Logical Operators

if (a > 0 && b > 0) { // AND: both must be true
    println("both positive")
}

if (a == 0 || b == 0) { // OR: at least one true
    println("at least one is zero")
}

if (!flag) { // NOT: inverts boolean
    println("flag is false")
}

Bitwise Operators

a = 0xFF & 0x0F    // AND: 15
b = 0x0F | 0xF0    // OR: 255
c = 0xFF ^ 0xAA    // XOR: 85
d = 1 << 3          // left shift: 8
e = 16 >> 2        // right shift: 4
f = ~0              // bitwise NOT: -1

Assignment Operators

x = 10      // simple assignment
x += 5     // x = x + 5
x -= 3     // x = x - 3
x *= 2     // x = x * 2
x /= 4     // x = x / 4
x &= 0xFF  // x = x & 0xFF
x |= 0x01  // x = x | 0x01
x ^= 0xAA  // x = x ^ 0xAA
x <<= 2    // x = x << 2
x >>= 1    // x = x >> 1

String Concatenation

The + operator concatenates strings. Non-string values on the right side are automatically converted to strings:

msg = "count: " + 42         // "count: 42"
msg = "pi = " + 3.14        // "pi = 3.14"
msg = "flag: " + true        // "flag: true"

Member Access

Dot Operator .

Accesses fields and methods on objects:

greeter.greet("World")
obj.field_name

Qualified Access ::

Accesses a specific base class member when there is ambiguity from multiple inheritance:

self.B::foo()   // call foo() from base class B

Automatic Semicolon Insertion

Nex inserts implicit statement terminators based on context. A newline terminates a statement if the previous token is one that can end a statement:

Inserts terminator afterDoes NOT insert after
Identifiers, literalsBinary operators (+ - * / %)

Explicit semicolons can be used to place multiple statements on one line:

a = 1; b = 2; c = a + b
