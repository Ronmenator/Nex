The Nex Programming Language

A compiled, statically-typed language with classes, interfaces, multiple inheritance, and native code generation. Designed for both rapid prototyping and production applications.

Static Typing

Type-safe by default with local type inference. Catch errors at compile time while writing concise code.

OOP First

Classes with real multiple inheritance, interfaces, structs, and virtual dispatch built into the core.

Native Performance

Compiles to native machine code via Cranelift — both AOT executables and JIT for scripting.

Rich Standard Library

15 built-in modules plus optional standard libraries for crypto, HTTP, UI, ML, and more.

Garbage Collected

Tracing GC manages memory automatically. Deterministic cleanup with using blocks.

Developer Friendly

REPL, formatter, linter, LSP, and VS Code extension included out of the box.

Quick Example

def factorial(n: Int) -> Int {
    if (n <= 1) {
        return 1
    }
    return n * factorial(n - 1)
}

def main() -> Unit {
    println("Factorial of 10:")
    println(factorial(10))
}
