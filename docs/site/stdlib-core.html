std.core

Built-in printing functions and the Object base class. Available without imports.

Auto-imported

Everything in std.core is automatically available in Every Nex program. You never need to write import std.core.

Functions

print(values...) → Unit

def print(values...) -> Unit

Prints one or more values to standard output without a trailing newline. Accepts String, Int, Double, Bool, and Char values. When multiple arguments are given, each is auto-converted to a string and joined with spaces (like Python's print).

print("Hello, ")
print("world!")
// Output: Hello, world!
print("value =", 42)
// Output: value = 42

println(values...) → Unit

def println(values...) -> Unit

Prints one or more values to standard output followed by a newline. Like print, multiple arguments are auto-converted and space-separated. Called with no arguments, println() prints a blank line.

println("Hello, world!")
// Output: Hello, world!

println(3.14)
// Output: 3.14

println(true)
// Output: true

Multi-argument printing

Pass multiple values separated by commas. Each value is auto-converted to a string and joined with spaces:

name = "Nex"
version = 1
pi = 3.14

println("Name:", name, "Version:", version, "Pi:", pi)
// Output: Name: Nex Version: 1 Pi: 3.14

String concatenation with +

You can also build strings manually with the + operator, which auto-converts non-string values:

println("Name: " + name)
println("Count: " + 42)

// Output:
// Name: Nex
// Count: 42

Object

Object — Universal Base Class

public class Object { }

Object is the universal base class for all class types in Nex. Every class implicitly derives from Object unless it explicitly declares other base classes (which themselves ultimately derive from Object).

Value Types

Value types (Int, Double, Bool, Byte, Char, Float, Int64, and struct types) do not inherit from Object. Only class types and String derive from Object.

// Every class implicitly derives from Object
class Animal {
    name: String
}

// Equivalent to:
class Animal : Object {
    name: String
}

// Derived classes form an inheritance chain through Object
class Dog : Animal {
    breed: String
}
