Control Flow

Conditionals, loops, and branching in Nex.

If / Else

The if statement evaluates a condition inside parentheses. The body is always a block with braces:

if (temperature > 30) {
    println("It's hot!")
}

If / Else If / Else

if (score >= 90) {
    println("A")
} else if (score >= 80) {
    println("B")
} else if (score >= 70) {
    println("C")
} else {
    println("F")
}

While Loop

Repeats a block while a condition is true:

i = 0
while (i < 10) {
    println(i)
    i = i + 1
}

For Loop

C-style for loop with three parts: initialization, condition, and step:

for (i = 0; i < 10; i += 1) {
    println(i)
}

You can also use var in the initializer for dynamic variables:

for (var i = 0; i < 5; i += 1) {
    println(i)
}

Break and Continue

break exits the innermost loop. continue skips to the next iteration:

i = 0
while (i < 100) {
    i = i + 1

    if (i % 2 == 0) {
        continue  // skip even numbers
    }

    if (i > 20) {
        break     // stop after 20
    }

    println(i)
}

Return

return exits the current function, optionally with a value:

def abs(n: Int) -> Int {
    if (n < 0) {
        return -n
    }
    return n
}

def greet() -> Unit {
    println("Hello!")
    return  // explicit return for Unit functions
}
